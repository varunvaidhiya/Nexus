# apps/extension

Chrome/Edge MV3 browser extension (Tier B) — captures web-chat conversations
(ChatGPT, Gemini, Claude.ai, DeepSeek, MiniMax, Qwen) on allowlisted domains
only, buffers in IndexedDB, and pushes to `/ingest`.

Built in **Phase 3** (see [docs/IMPLEMENTATION_PLAN.md](../../docs/IMPLEMENTATION_PLAN.md)).
